### 它是什么？

Lite分支 是HotUP的分支版本，是脱离Mount挂载的纯Systemless挂载的温控移除模块

### 有什么用？
某些设备无法正常使用Mount挂载，或有特殊需求，故有了此版本，如果没有什么特殊情况可以选择原版[HotUP](./../)

### 环境需求：
诶嘿，没有环境需求，这是一个全系统通用的温控移除模块
