HotUP：Thermal_Checker
:::

### 它是什麽？

Thermal_Checker 是HotUP的附屬項目，是為 HotUP 編寫的具有寬容性的溫控檢測脚本

### 有什么用？

Thermal_Checker 不光可以用于检测使用 HotUP 后是否正常移除温控，还可以检测其他温控移除模块是否正常移除，是通用的温控检测脚本
