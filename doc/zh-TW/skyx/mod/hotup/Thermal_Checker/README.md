<div align="center">HotUP：Thermal_Checker</div>

### 它是什么？

Thermal_Checker 是HotUP的附属项目，是为 HotUP 编写的具有宽容性的温控检测脚本

### 有什么用？

Thermal_Checker 不光可以用于检测使用 HotUP 后是否正常移除温控，还可以检测其他温控移除模块是否正常移除，是通用的温控检测脚本
