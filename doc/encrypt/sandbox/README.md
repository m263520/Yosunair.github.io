尤苏奈尔基本计划：Sandbox
:::

### 它是什么？
Sandbox 是一个基于[开源加固方案](https://github.com/Frezrik/Jiagu)的修改版本的加固，使用了[StringFog](https://github.com/MegatronKing/StringFog)加密和非常惹眼的混淆功能，并且添加了jni层的签名校验和软件入口的校验，提高安全性
